# Dotfiles

One single repo to rule 'em all!

## Quickstart

Install:

    $ git clone git://github.com/iromli/dotfiles.git --recursive
    $ cd /path/to/dotfiles
    $ fab install

Uninstall:

    $ cd /path/to/dotfiles
    $ fab remove
